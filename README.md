Ứng dụng giúp người dùng **quản lý tài chính cá nhân** một cách trực quan và hiệu quả.  
Được xây dựng bằng **Flutter** với giao diện hiện đại và tích hợp **Firebase Cloud**.

---

## ✨ Tính năng nổi bật

- 📊 **Dashboard trực quan**  
  Theo dõi tổng chi tiêu hàng tháng với biểu đồ sinh động.

- 💰 **Quản lý giao dịch**  
  Thêm / xóa / phân loại chi tiêu nhanh chóng.

- 📈 **Phân tích chi tiết**  
  Biểu đồ tròn giúp hiển thị tỷ lệ chi tiêu theo danh mục.

- 🎯 **Lập kế hoạch tài chính**  
  Thiết lập hạn mức để kiểm soát ngân sách.

- 🔥 **Thử thách tiết kiệm**  
  Ví dụ: *“Cai nghiện Shopee”* giúp hình thành thói quen tốt.

- 🎨 **UI hiện đại**  
  Hỗ trợ **Dark Mode / Light Mode** + Glassmorphism.

---

## 🛠 Công nghệ sử dụng

| Công nghệ | Mô tả |
|----------|------|
| Flutter | Xây dựng UI đa nền tảng |
| Firebase Auth | Xác thực người dùng |
| Cloud Firestore | Lưu trữ dữ liệu |
| Firebase Storage | Lưu trữ file |
| Google Cloud | Backend |
