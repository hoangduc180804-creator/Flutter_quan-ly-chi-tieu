A mobile application that helps users **manage personal finances** in an intuitive and efficient way.  
Built with **Flutter**, featuring a modern UI and integrated with **Firebase Cloud services**.

---

## ✨ Key Features

- 📊 **Interactive Dashboard**  
  Track monthly expenses with dynamic charts and insights.

- 💰 **Transaction Management**  
  Quickly add, delete, and categorize expenses.

- 📈 **Detailed Analytics**  
  Pie charts visualize spending distribution by category.

- 🎯 **Financial Planning**  
  Set spending limits to better control your budget.

- 🔥 **Saving Challenges**  
  Features like *“No Shopping Challenge”* help build better spending habits.

- 🎨 **Modern UI Design**  
  Supports **Dark Mode / Light Mode** with Glassmorphism style.

---
